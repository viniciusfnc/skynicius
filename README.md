# skynicius
Challenge - flight price.


## Main Paths
http://localhost:8080/swagger-ui/#
- **/flights/avg -**  *Returns average flight prices (GET)*
- **/flights -**  *Find all flight summaries (GET)*
- **/flights -** *Delete all flight summaries - (DELETE)*
